# YoutubeLiveWidget
サポーターズハッカソンVol.11の制作物になります。

## アプリの説明
- スマホ(Android)のホームウィジェット
- Youtubeのライバーの配信状態を取得して配信していればアイコン画像をクリックすると配信動画を開けるアプリ
